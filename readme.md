# POS Cafeteria

Sistema de Punto de Venta (POS) diseñado para gestionar pedidos y ventas en una cafetería de manera eficiente.

## Demo de la pagina

https://pos-cafeteria.netlify.app/

## Herramientas y Tecnologías Utilizadas

- **Lenguaje**: Python 3.x
- **Framework**: Flask para el backend web.
- **Base de Datos**: SQLite para almacenamiento local.
- **Frontend**: HTML/CSS/JavaScript con Bootstrap.
- **Control de Versiones**: Git.
- **Entorno de Desarrollo**: Visual Studio Code.

## Despliegue

1. Clona el repositorio: `git clone https://github.com/usuario/pos-cafeteria.git`
2. Instala dependencias: `pip install -r requirements.txt`
3. Ejecuta la aplicación: `python app.py`
4. Accede en el navegador a `http://localhost:5000`

## Uso

1. Inicia sesión con credenciales de administrador.
2. Agrega productos al menú desde la interfaz.
3. Registra pedidos de clientes.
4. Genera reportes de ventas diarios.
