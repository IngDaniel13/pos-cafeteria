# POS Cafeteria

Sistema de Punto de Venta (POS) diseñado para gestionar pedidos y ventas en una cafetería de manera eficiente, utilizando una interfaz web simple y una base de datos en la nube.

## Demo de la pagina

https://pos-cafeteria.netlify.app/

## Herramientas y Tecnologías Utilizadas
- **Supabase** (Base de datos en la nube)
- **HTML/CSS/JavaScript** (Desarrollo frontend)
- **Git** (Control de versiones)

## Despliegue

1. Clona el repositorio: `git clone https://github.com/usuario/pos-cafeteria.git`
2. Abre el archivo `index.html` en tu navegador web para ejecutar localmente.
3. Para desplegar en Netlify: Sube los archivos al repositorio y conecta con Netlify para despliegue automático.

## Uso

1. Inicia sesión con credenciales de administrador.
2. Agrega productos al menú desde la interfaz.
3. Se debe registrar la cantidad de productos del dia en el apartado de "Produccion"
4. Luego en el boton "POS" aparecen los productos que se registraron y su cantidad.
